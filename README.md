# Client CRUD
 
Essa API para cadastro de clientes foi desenvolvida utilizando Flask, o template "Argon Dashboard" , SQLAlchemy e muita dedicação. =)

## Como executar
Na primeira vez:
```
pip install -r requirements.txt
```
A partir de então:

Para Windows:
```
set FLASK_APP=app
flask run
```

Para MAC e Linux:
```
export FLASK_APP=app
flask run
```
